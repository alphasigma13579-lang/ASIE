# نموذج دراسة جدوى تفصيلي معبأ — ASIE

هذا النموذج يوضح شكل التقرير النهائي قبل التنفيذ، اعتماداً على بنية النماذج المرفقة: السوق، المنافسون، المنتجات، الدراسة الفنية، التشغيل، التسويق، التأسيس، التشغيل، التمويل، القوائم المالية، التدفق النقدي، الحساسية، SWOT، المخاطر، والجاهزية التمويلية.

## طبيعة الأرقام

الأرقام في النموذج افتراضية ومترابطة لغرض اختبار شكل التقرير وآلية اتخاذ القرار. لا تمثل بيانات مشروع حقيقي، ولا عرض تمويل، ولا قبولاً من بنك أو صندوق أو مستثمر. يجب استبدالها بعروض أسعار، عقود، قوائم، كشوف، تراخيص، ومسوح سوق قبل أي تقديم خارجي.

## المشروع الافتراضي

- النشاط: مقهى ومخبز مختص.
- الموقع: حي تجاري في الرياض، يحتاج إلى عقد أو خطاب نوايا.
- الاستثمار: 1,300,000 ريال.
- التمويل المقترح: 450,000 ريال مساهمة مالك و850,000 ريال تمويل طويل الأجل.
- التوقع: 2027–2031.
- المخرج: PDF عربي تفصيلي من 25 صفحة.

## ملف PDF

المخرج القابل للمعاينة موجود في:

`output/pdf/detailed-feasibility-study-example.pdf`

وتم توليده من:

`tools/generate_detailed_feasibility_report.py`

## بوابات استبدال الافتراضات

قبل اعتبار التقرير صالحاً للتقديم، يجب إغلاق بوابات الموقع، الطلب، عروض الأسعار، التراخيص، التمويل، المحاسبة، والمراجعة البشرية. بعدها يُعاد بناء التقرير من Snapshot المشروع الفعلي، ولا تُستخدم هذه العينة كما هي.
