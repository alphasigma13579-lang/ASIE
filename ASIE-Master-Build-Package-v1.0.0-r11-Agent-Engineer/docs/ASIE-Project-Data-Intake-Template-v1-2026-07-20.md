## 3. السوق والطلب

- من هو العميل الأساسي؟
- كم عدد العملاء أو الطلبات المتوقع يومياً/شهرياً؟ وما مصدر التقدير؟
- من أهم خمسة منافسين؟
- ما أسعارهم ومواقعهم ونقاط قوتهم؟
- هل توجد عقود، طلبات شراء، خطابات اهتمام، أو مبيعات تاريخية؟
- ما القنوات: موقع، متجر إلكتروني، تطبيقات، موزعون، عقود شركات؟

## 8. البيانات المالية التاريخية

إذا كان المشروع قائماً، أرفق آخر 12–24 شهراً:

- المبيعات الشهرية.
- تكلفة المبيعات.
- الرواتب والإيجار والمصروفات.
- كشوف البنك.
- الذمم والمخزون والالتزامات.
- قائمة الدخل والميزانية والتدفق النقدي إن وجدت.

## 10. المستندات المطلوبة

- الهوية/السجل التجاري وعقد التأسيس.
- عقد الموقع أو خطاب النوايا.
- عروض أسعار التأسيس والمعدات.
- التراخيص والموافقات.
- بيانات السوق والمنافسين.
- العقود أو خطابات الاهتمام.
- كشوف الحساب والقوائم المالية إن كان المشروع قائماً.
- عروض التمويل وشروط الدين.
- أي دراسة سابقة أو خطة تشغيل.

## قاعدة الاستخدام

لا تُخمن أي رقم مؤثر تمويلياً. كل رقم نهائي في تقرير ASIE يجب أن يُصنف كواحد من: **دليل موثق، افتراض معلن، قيمة مشتقة، أو فجوة تحتاج إغلاقاً**.
