# ASIE — مخطط تصحيح تجربة العميل إلى مستشار مشروع

## القرار

واجهة العميل ليست Console ولا أرشيف تقارير. هي **مستشار تنفيذ مشروع**: يبدأ بالنتيجة العملية، ثم يشرح ما يلزم فعله اليوم، وما الذي يحتاجه لاتخاذ قرار أقوى.

لا تعرض الواجهة الخوارزميات أو الـRuntime أو معرفات العقود. تبقى هذه للمشغل والتشخيص فقط.

## صفحة العميل الرئيسية

```
قرار مشروعي الآن
├─ هل أمضي / أراجع / أتوقف؟ ولماذا؟
├─ 3 خطوات عملية تالية مرتبة بالأثر
├─ التمويل والدعم: ما الاحتياج؟ وما المسار المناسب؟
├─ السوق والمنافسون: ما ثبت محلياً وما ينقص دليلاً
├─ الموقع: نطاق المشروع، تغطية البيانات، وتقييم المواقع المدخلة
├─ التراخيص والإجراءات: checklist حسب النشاط والموقع
├─ مشاريع مشابهة: كتالوج موافق عليه أو عينات موسومة فقط
└─ تقارير وSnapshot: ملف القرار القابل للمشاركة
```

## ما يمكن عرضه الآن بصدق

| التجربة | مصدر الحقيقة الحالي | الواجهة المقترحة |
|---|---|---|
| القرار والتمويل والمخاطر | Snapshot وDecision Pack | ملخص تنفيذي وخطوة تالية وتقرير PDF/HTML محلي |
| فجوات الأدلة والمنافسة | `sector_evidence_map` و`competition_intensity` عند ربط دليل | «ما نعرفه / ما ينقصنا» بلا ادعاء معرفة منافسين |
| التراخيص والدعم | checklist محلي يملؤه المشغل أو العميل مع مصدر/تاريخ مراجعة | مسار إجراء، لا فتوى أو رابط حكومي موهم |
| الموقع | نطاق يحدده العميل وdataset مكاني محلي/مرخص يرفعه | خريطة بيانات محلية فقط، مع تغطية ودقة واضحة |
| المشاريع المشابهة | كتالوج مستقل بموافقة صريحة أو بيانات مثال | لا يُعرض أي مشروع منظمة أخرى تلقائياً |

## ما لا يجوز ادعاؤه قبل بناء سجله

- منافسون في مدينة/حي/نطاق محدد.
- أفضل شارع أو موقع افتتاح.
- أهلية تأشيرة أو ترخيص أو دعم حكومي.
- مشروع مشابه مأخوذ من منظمة أخرى.
- خريطة حية أو بيانات سوق خارجية.

كل واحدة تتطلب سجلاً خلفياً محكوماً، مصدره، تاريخ مراجعته، نطاقه الجغرافي، مستوى الثقة، وحق عرضه.

## عقود البناء التالية

1. **Local Market Evidence Pack:** منافس/موقع/منطقة/صفة/مصدر/تاريخ/دقة/حالة مراجعة؛ upload محلي فقط.
2. **Guidance Catalog:** دعم/ترخيص/تأشيرة/خطوة/مالك/رابط مرجعي محلي/تاريخ مراجعة/تنبيه قانوني؛ لا تنفيذ أو جلب خارجي.
3. **Site Candidate Evaluation:** مواقع يدخلها العميل أو المشغل، مع معايير شفافة مرتبطة بدليل؛ لا توصية «أفضل موقع» بلا بيانات كافية.
4. **Consent-based Similar Projects Catalog:** حالات منشورة بموافقة، أو عينات Demo موسومة؛ لا cross-tenant discovery.
5. **Client Advisor UI:** تبويبات بسيطة: `قراري`، `خطتي`، `السوق والموقع`، `الدعم والإجراءات`، `تقاريري`.

## قاعدة القرار

إذا لم يوجد دليل محلي أو مصدر مرخص ومراجع، تعرض ASIE حالة **«نحتاج بيانات»** وخطوة لجمعها. لا تستبدل غياب الحقيقة بخريطة جميلة أو توصية تخمينية.

### استثناء بيئة التطوير المحلية

يجوز استخدام بيانات منافسين ومواقع وإشارات سوق **تجريبية موسومة `DEMO / LOCAL ONLY`** لبناء الواجهة واختبار التفاعل قبل وجود بيانات حقيقية. لا تدخل هذه البيانات في Snapshot حقيقي أو تقرير عميل أو قرار، وتزال أو تستبدل بسجلات موثقة قبل أي إطلاق خارجي.

## بوابة التنفيذ

يتطلب التنفيذ موافقة صريحة على هذا التوسع لأنه يضيف نماذج بيانات وتجارب قرار جديدة، مع بقاء الشبكة الخارجية معطلة وAAS Runtime Freeze بلا تعديل.
