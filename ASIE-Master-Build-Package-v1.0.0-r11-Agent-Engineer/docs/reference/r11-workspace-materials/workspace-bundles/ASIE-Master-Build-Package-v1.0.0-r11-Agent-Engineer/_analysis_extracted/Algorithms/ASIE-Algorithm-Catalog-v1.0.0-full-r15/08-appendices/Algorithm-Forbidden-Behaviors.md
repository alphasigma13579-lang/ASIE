# Algorithm Forbidden Behaviors

## السلوكيات الممنوعة في الخوارزميات

1. AI generates financial numbers.
   الذكاء الاصطناعي يولد أرقامًا مالية.

2. AI fills missing market evidence.
   الذكاء الاصطناعي يعوض الأدلة السوقية الناقصة.

3. Finance Engine calls market providers directly.
   محرك التمويل يتصل بمزودي السوق مباشرة.

4. Market Intelligence performs finance formulas.
   ذكاء السوق ينفذ معادلات التمويل.

5. Decision Council approves without Validation Gate.
   مجلس القرار يوافق بدون بوابة تحقق.

6. Admin silently overrides policy.
   الإدارة تتجاوز سياسة بصمت.

7. MCMC runs without fixed seed.
   MCMC يعمل بدون بذرة ثابتة.

8. Algorithm ownership changes without ACP.
   ملكية الخوارزمية تتغير بدون ACP.

9. User is required to fetch public evidence manually.
   إلزام المستخدم بجلب الأدلة العامة يدويًا.

10. Deep research output has no source URL.
   مخرج البحث العميق بلا رابط مصدر.

11. Pinecone is treated as source of truth.
   التعامل مع Pinecone كمصدر حقيقة.

## Advanced Feasibility Forbidden Behaviors r15

- Selecting or downgrading a study profile with AI.
- Marking a mandatory chapter complete without owner output, evidence, assumptions, validation and reconciliation.
- Creating finance inputs from SWOT, PESTEL, Porter, BMC, VPC, narrative text, screenshots or commercial references.
- Using plug figures to balance statements without an explicit approved account and traceable basis.
- Mixing currencies, periods, real/nominal bases, pre/post-tax bases, or project/equity/economic perspectives silently.
- Selecting one IRR root silently or hiding that no valid root exists.
- Using MCMC without fixed seed, approved ranges/distributions, correlation controls and full-model recalculation.
- Letting AI choose financial/economic inputs or interpret simulation as a guaranteed outcome.
- Treating general MOF/Etimad references as exact competition controls or universal feasibility approval.
- Automated Aljdwa access, copying, summarization, RAG, embedding, indexing, reconstruction or monitoring without exact written permission.
