# Sector Taxonomy and Investment Signals

## تصنيف القطاعات وإشارات الاستثمار

## Purpose

## الهدف

Define ASIE's official sector taxonomy for Saudi Arabia, GCC, and Middle East business analysis.

تحديد تصنيف القطاعات الرسمي في ASIE لتحليل الأعمال في السعودية والخليج والشرق الأوسط.

In v1, project analysis remains Saudi Arabia only. GCC and Middle East sectors are used for benchmarking, source mapping, and strategic context.

في النسخة الأولى، تحليل المشروع يبقى داخل السعودية فقط. قطاعات الخليج والشرق الأوسط تستخدم للمقارنة وربط المصادر والسياق الاستراتيجي.

## Taxonomy Rule

## قاعدة التصنيف

Every project must map to:

كل مشروع يجب أن يرتبط بـ:

- Primary sector / القطاع الرئيسي.
- Subsector / التصنيف الفرعي.
- Activity classification / تصنيف النشاط.
- Investment indicators / مؤشرات الاستثمار.
- Evidence sources / مصادر الأدلة.
- Evaluation criteria / معايير التقييم.

## Sector Evaluation Criteria

## معايير تقييم القطاع

Each sector must define relevant criteria. KIMI must not apply one generic scoring model to all sectors.

كل قطاع يجب أن يحدد معايير ملائمة. ممنوع على KIMI استخدام نموذج تقييم عام واحد لكل القطاعات.

Common criteria:

معايير عامة:

- Market size / حجم السوق.
- Growth trend / اتجاه النمو.
- Regulation intensity / كثافة التشريعات.
- Capital intensity / كثافة رأس المال.
- Labor dependency / الاعتماد على العمالة.
- Import dependency / الاعتماد على الاستيراد.
- Location sensitivity / حساسية الموقع.
- Price volatility / تذبذب الأسعار.
- Competition intensity / شدة المنافسة.
- Vision 2030 alignment / التوافق مع رؤية 2030.
- Sustainability impact / أثر الاستدامة.
