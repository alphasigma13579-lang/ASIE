## نطاق النسخة الأولى

| Decision | Arabic | Value |
| --- | --- | --- |

## MVP Modules

## موديولات النسخة الأولى

1. `User / Auth Module` / موديول المستخدمين والمصادقة.
2. `Project Wizard Module` / موديول معالج إنشاء المشروع.
3. `Market Intelligence Module` / موديول ذكاء السوق.
4. `Finance Engine Module` / موديول محرك التمويل الحتمي.
5. `AI Advisory Module` / موديول الاستشارة بالذكاء الاصطناعي.
6. `Decision Council Module` / موديول مجلس القرار.
7. `Reports Module` / موديول التقارير.
8. `Admin Module` / موديول الإدارة.
9. `Audit / Observability Module` / موديول التدقيق والمراقبة.
10. `Subscription / Usage Module` / موديول الاشتراكات والاستخدام.

## Explicitly Deferred

## مؤجل صراحة

- Multi-country analysis / التحليل متعدد الدول.
- Government connector automation beyond documented contracts / أتمتة الربط الحكومي خارج العقود.
- Open plugin marketplace / متجر إضافات مفتوح.
- Fully autonomous AI decisions / قرارات AI ذاتية كاملة.
- Treating legacy screenshots as production data / اعتبار صور النسخة القديمة بيانات إنتاج.
- Claiming government approval from an internal ASIE model or analytical framework / ادعاء اعتماد حكومي لنموذج داخلي أو إطار تحليلي.
