# ASIE r11 — Workspace Reference Materials

هذه الحزمة تحفظ الخطط والوثائق المرجعية التي كانت موجودة في مساحة العمل خارج مجلد البناء الرئيسي، وتُرفع مفكوكة لتكون قابلة للبحث والمراجعة داخل GitHub.

## قواعد المصدر

- وثائق `docs/` الحالية داخل المشروع هي مصدر الحقيقة التشغيلي الأحدث.
- هذه الملفات تمثل خططاً ومواد مرجعية وسجلات عمل سابقة، ولا تستبدل AIA-01 أو AIA-02 v1.2.1.
- الملفات المضغوطة، الصور، سجلات التشغيل، قواعد البيانات، وملفات Python المولدة مستبعدة من النسخة المفكوكة.
- لا تمنح هذه المواد صلاحية إطلاق إنتاجي أو تفعيل شبكة أو مزودين أو AI.

## provenance

- المصدر الأصلي: مساحة عمل ASIE r11 المحلية.

- المراجعة: مرجع عمل؛ يلزم اعتماد صريح قبل اعتباره سياسة تشغيلية نافذة.
